### ✅ 合理位置判斷條件

1. **📍 有效座標**: 經緯度不為空值 (NaN)
2. **🎯 非原點**: 座標不為 (0, 0)
3. **🌍 邊界內**: 在台灣地理邊界範圍內
4. **🏢 陸地區域**: 在台灣主要地區範圍內

### ❌ 不合理位置類型定義

1. **🚫 空值座標**: 經度或緯度為空值
2. **🎯 原點異常**: 座標為 (0, 0)
3. **🌊 海上點位**: 在台灣邊界內但不在陸地區域
4. **🚫 邊界外異常**: 超出台灣地理邊界

## 🛠️ 技術實現

### 🔧 核心判斷函數

```python
def check_location_reasonability(lat, lon):
    """檢查避難所位置是否合理"""
    
    # 檢查是否為有效座標
    if pd.isna(lat) or pd.isna(lon):
        return False
    
    # 檢查原點 (0,0)
    if lat == 0 and lon == 0:
        return False
    
    # 檢查是否在台灣邊界內
    if not (taiwan_boundaries['min_lon'] <= lon <= taiwan_boundaries['max_lon'] and
            taiwan_boundaries['min_lat'] <= lat <= taiwan_boundaries['max_lat']):
        return False
    
    # 檢查是否在任何合理的台灣地區範圍內
    for region_name, region_bounds in taiwan_regions.items():
        if (region_bounds['min_lon'] <= lon <= region_bounds['max_lon'] and
            region_bounds['min_lat'] <= lat <= region_bounds['max_lat']):
            return True
    
    # 如果不在任何合理範圍內，則認為是海上點位
    return False
```
